Author: Yoann Guyot (yoann.guyot@cetic.be) contractor for Alstom

This directory contains some first tries to model the EVC kernel using Papyrus SysML models and ProR requirements.

This is an Eclipse project. It can be imported using the import function of Eclipse:
File > Import... > General > Existing Projects into Workspace > Next > Select root directory > Browse... > EVC-Kernel_Model > OK > Copy projects into workspace > Finish
