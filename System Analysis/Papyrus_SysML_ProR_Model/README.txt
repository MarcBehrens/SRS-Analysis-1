Author: Yoann Guyot (yoann.guyot@cetic.be) contractor for Alstom

This directory contains some first tries to model SRS analysis using
Papyrus SysML models and ProR requirements.

The SRS_Analysis directory contains an Eclipse project. It can be
imported using the import function of Eclipse:
File > Import... > General > Existing Projects into Workspace > Next >
Select root directory > Browse... > SRS_Analysis > OK > Copy projects
into workspace > Finish
